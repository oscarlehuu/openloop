# Code of Conduct

## Standard

OpenLoop uses a simple contributor standard: be respectful, constructive, and focused on the work. Do not use harassment, threats, personal attacks, sexualized language, or discriminatory behavior.

## Scope

This applies to GitHub issues, pull requests, discussions, code review, and any project-maintainer communication related to OpenLoop.

## Reporting

Report unacceptable behavior to the project maintainer through a private GitHub security advisory when privacy matters, or by contacting `@oscarlehuu` on GitHub.

## Enforcement

The maintainer may remove comments, close issues, reject pull requests, or block participants who violate this code of conduct.
